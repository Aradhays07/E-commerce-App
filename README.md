## Installation

Build E-commerce App from the source and install dependencies:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Aradhay07/E-commerce-App
   ```

2. **Navigate to the project directory**:
   ```bash
   cd E-commerce-App
   ```

3. **Install the dependencies**:
   ```bash
   npm install
   ```
